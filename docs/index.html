Three actions. One key.

NothingHere performs three critical actions simultaneously the moment you press your hotkey.

Hide All Windows

Instantly minimizes or hides every visible window across all applications. Your screen goes clean in milliseconds.

Mute Sound

Mutes all system audio instantly. Music, podcasts, videos — everything goes silent simultaneously.

Open Cover Document

Launches your pre-configured cover file — a spreadsheet, document, or anything that looks like real work.

How it works

Set up once, use forever. Three simple steps to total screen privacy.

1

Set your hotkey

Choose any key combination. ⌘ ⇧ H or whatever feels natural.

2

Configure your cover

Pick a document, spreadsheet, or any file that looks like legitimate work.

3

Press and relax

One key press hides everything, mutes sound, and opens your cover. Done in milliseconds.

Guard Mode

Guard Mode lives in your menu bar. When armed, NothingHere watches for any key press and is ready to act instantly.

Toggle between armed and disarmed states with a single click. The menu bar icon changes to reflect the current state — so you always know if you're protected.
